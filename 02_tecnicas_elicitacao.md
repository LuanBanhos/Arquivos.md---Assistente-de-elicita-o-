# Técnicas de Elicitação

## Entrevistas

Podem ser estruturadas ou semiestruturadas.

São úteis para compreender:

- processos de trabalho
- dificuldades enfrentadas pelos usuários
- necessidades operacionais
- expectativas sobre o sistema

## Questionários

São úteis quando há muitos participantes.

Permitem:

- padronizar respostas
- coletar opiniões em larga escala
- identificar padrões de necessidade

## Observação

Consiste em observar o usuário executando suas atividades.

É útil para identificar:

- requisitos implícitos
- processos informais
- comportamentos que não são mencionados nas entrevistas

## Uso dessas técnicas na análise

Ao analisar uma entrevista ou questionário:

- identificar necessidades expressas pelos usuários
- reconhecer possíveis requisitos implícitos
- registrar evidências diretamente do texto analisado
